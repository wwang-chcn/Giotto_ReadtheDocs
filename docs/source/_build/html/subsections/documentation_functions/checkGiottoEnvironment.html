Check the Giotto Environment¶

checkGiottoEnvironment()

Removes the Giotto environment

checkGiottoEnvironment(verbose = TRUE)

Arguments¶

verbose

be verbose

Details¶

Checks if a miniconda Giotto environment can be found. A miniconda Giotto environment can be installed with installGiottoEnvironment().
